    How it works

    Bayesian AI (default, strongest). Maintains a persistent population of thousands of "particles" — complete, internally consistent guesses at the whole fleet's layout — across the entire game, filtering (not rebuilding) it after every shot. A particle survives a miss only if none of its ships are there, and survives a hit only if one of its ships is. Cells are scored from what fraction of surviving particles agree on them, split into two questions: is anything here at all (occupancy), and does a ship we've already found but not finished continue into here (active-ship / sink probability). Once any hits are on the board, the second question dominates the score, so it finishes ships it's found before hunting new ones. Retaining each particle's individual ship identities — instead of collapsing a particle into a single "occupied cells" set — is what lets it reason correctly when two ships happen to be placed touching each other: it never has to decide which specific ship a run of hits belongs to, since every consistent interpretation stays alive in the population until later shots naturally rule the wrong ones out.

    Probability AI (heuristic). A much faster approximation: for every ship length in the fleet, it enumerates every placement still consistent with the board and tallies how many valid placements cover each cell, independently per ship length. A placement only counts toward "finishing a found ship" if it still has an unknown cell to give (not if it's already fully revealed) — good enough to avoid wasting shots on a fully-explained run, but unlike the Bayesian AI it scores each ship length in isolation, so it can't fully rule out an option that looks locally plausible but is actually impossible once the rest of the board is taken into account.

    The enemy fleet you attack is placed by the same style of algorithm in reverse: it plays many simulated games against the Bayesian AI for a batch of random legal layouts and keeps whichever one survives the longest on average — the "Auto-Place (Smart)" button uses the same search for your own fleet.
